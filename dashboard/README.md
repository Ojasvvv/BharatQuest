# Dashboard

React + Vite + TailwindCSS + Recharts frontend for the Apatheia sandbox dashboard.

## Setup (Phase 5+)

```bash
cd dashboard
npm create vite@latest . -- --template react-ts
npm install tailwindcss @tailwindcss/vite recharts
npm run dev
```

## Features (Planned)
- Real-time execution metrics visualization (Recharts)
- Code editor with JS syntax highlighting
- Live WebSocket output streaming
- Latency breakdown charts (instantiation vs eval vs total)
